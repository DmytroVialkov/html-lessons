• Домашка до уроку №2

    1. Вивчити пройдені HTML-теги
    2. Зверстати статтю про себе (або будь-яку іншу статтю), використавши УСІ вивчені теги,
    3. ОБОВ'ЯЗКОВО перед здачею ДЗ користуємось сервісом для перевірки вкладеності https://caninclude.glitch.me/ та перевіряємо верстку у валідаторі https://validator.w3.org/.
    4. Завантажити домашку на https://github.com/
    5. Відправляємо посилання на домашку у "показ-домашок -> Урок №2"

  • Домашка до уроку №3

    1. видаляємо плагін редактора prettier
    2. Вивчити атрибут class та правила імен класів
    3. Вивчити методи підключення CSS та їх різницю,
    4. Створюємо та підключаємо css файл, додаємо файл обнулення стилів.
    5. Верстаємо статтю про Україну з посиланнями на розділи, використовуючи отримані знання про HTML-теги та вкладеність (коробочки) та призначення тегів. До статті має бути під'єднаний файл стилів та файл обнулення стилів.
    6. Практикуємо КОЖЕН з вивчених селекторів, комбінацій та вкладеність. Використовуємо ЛИШЕ CSS-властивість color. Експерементуємо!
    7. Обов'язково користуємось валідаторами:
      • html-валідатор
      • css-валідатор
      • валідатор вкладення
      Додатково Emmet:
      • https://docs.emmet.io/
      • https://epixx.github.io/emmet/
      Додатково відео:
      • https://www.youtube.com/watch?v=UOgtWxnv92c

  • Домашка до уроку №4

    1. Вантажимо курсові макети у FIGMA (https://www.figma.com/).
    2. Знайомимось з основними панелями сервісу
    3. Верстаємо нову статтю, використовуючи УСІ властивості шрифту та тексту та їх значення.
    4. Верстаємо блок з макету Ecobazar (виділено на скріні). Підключаємо шрифти (2-3 родини та 2-3 варіаціях в кожній родині) УСІМА методами. (як з Google так і файлами)
    5. Сервіси:
      • https://cloudconvert.com/ttf-to-woff2
      • https://fonts.google.com/
