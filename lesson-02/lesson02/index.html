↑ Структурні теги

↑ HEADER

Елемент заголовка представляє групу вступних або навігаційних засобів.

Елемент заголовка зазвичай має містити заголовок розділу (елемент h1-h6 або елемент hgroup), але це не є обов'язковим. Елемент заголовка також може бути використаний для загортання змісту розділу, форми пошуку або будь-якого відповідного логотипу.

Елемент заголовка не розділяє вміст; він не вводить новий розділ.

↑ FOOTER

Елемент підвалу представляє підвал для свого батьківського елемента або секції, якщо він є останнім дочірнім елементом.

Елемент підвалу зазвичай містить авторські дані, правову інформацію або посилання на додаткові ресурси.

↑ MAIN

Елемент main представляє головний контент документа. Головний контент документа включається в елемент main, який не повинен містити елементів header, footer, або nav.

Елемент main не повинен бути вкладений в елемент article, aside, footer, header або nav. Якщо він використовується, він повинен бути вкладений в елемент body.

↑ DIV

Елемент div представляє контейнер для елементів блоку. Він не має ніякого семантичного значення.

Елемент div часто використовується для групування елементів блоку з метою стилізації їх за допомогою CSS або визначення їхньої поведінки за допомогою скриптів.

↑ Контентні теги

↑ H1

Елемент заголовка h1 представляє найважливіший заголовок першого рівня.

↑ H2—H6

Елементи заголовка h2-h6 представляють заголовки другого, третього, четвертого, п'ятого і шостого рівнів відповідно.

↑ P

Елемент абзацу p представляє абзац тексту.

↑ MARK

Елемент mark представляє текст, який позначений або виділений для пошуку або посилання.

↑ SPAN

Елемент span представляє контейнер для текстового контенту, який не має семантичного значення.

↑ BR

Елемент br представляє розрив рядка.

↑ UL

Елемент ненумерованого списку ul представляє ненумерований список елементів.

↑ OL

Елемент нумерованого списку ol представляє нумерований список елементів.

↑ LI

Елемент списку li представляє елемент списку.

↑ IMG

Елемент img представляє зображення.

↑ A

Елемент a представляє гіперпосилання.
